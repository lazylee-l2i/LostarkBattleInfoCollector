# LostarkBattleInfoCollector
 로스트아크 영지 관련 아이템 크롤러

# 개발환경
1. OS : Windows 10
2. Python : 3.9.x

 ## 워크로드
 ```
1. Inven or LostArk Official Web 크롤링
    1.1 배틀아이템
        - 이름
        - 효과
        - 제작 가격
        - (제작 재료)
    1.2 생활 재료
        - 이름
        - 사용처
        - 경매장 가격
 ```
